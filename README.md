# Schotten Totten - C++ Project

This project is a C++ implementation of the game **Schotten Totten**.  
It was developed collaboratively with **Nassim BENALAYA** and **Lucas JUSSERAND**.

## Description

Schotten Totten is a strategic card game where two players compete to take control of nine stones placed between them. This project implements:  

- Game logic (rules, victory conditions, etc.)
- Player and hand management
- Console-based game board display
- Player interactions

## Features

- Automatic detection of victory conditions
- Clear display of the board and cards
- Local two-player mode
- Complete management of game turns and card placement
